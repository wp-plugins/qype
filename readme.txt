Display qype-reviews in the sidebar. Highly customizable.

== Description ==
The plugin provides a sidebar widget, that displays qype-reviews (http://www.qype.com) of a
certain place. Qype is a social network to rate pubs and bars around the world, specially in europe.

It is highly customizable: You can decide, which ratings are displayed, i.e. how many rating point 
does a review need at minimum, to be displayed, e.g. reviews with 4 or 5 stars only.

You can also edit the preferences about username, user-image, date and link to qype.


== Installation ==

1. The plugin is available for download on the WordPress repository: http://wordpress.org/extend/plugins/qype/
2. Download the file `qype.zip`, and uncompress it,
3. Upload the resulting files to a folder in `../wp-content/plugins/`,
4. Activate WP Qype through the 'Plugins' menu in WordPress,
5. The plugin is ready to be used.

Hint: 
Alternativley you can use the "One Click Plugin Updater"
http://wordpress.org/extend/plugins/one-click-plugin-updater/


== Usage ==

* Go to the **Appearence / Widgets** menu, 
* Active the widget named: WP Qype by dragging and dropping it on the sidebar-section
* Configure it, through the widget menu.
